# Restaurant Management System

This project is a unified restaurant management system developed as an Android mobile application as part of my "Restaurant Management System" bachelor's thesis. It is designed to support both employee and customer workflows and provides a complete set of tools for managing restaurant operations, including:

* Placing and managing orders from both staff and customer sides  
* Editing dishes and modifying the menu  
* Monitoring ingredient stock levels and warehouse status  
* Assigning discounts, rewards, and customer benefits  
* Facilitating communication between the restaurant and its customers  

The system is backed by Firebase Cloud Firestore (NoSQL) and Firebase Authentication, ensuring secure data handling and real-time synchronization across all users.

This application is written in **Kotlin** for Android devices and is based on **Android SDK 33** and **JDK 17**.

> **Note:**  
> All Firebase Authentication providers and Firestore access are currently **disabled**.  
> To run the application, please contact the author or create your own Firebase project and configure the necessary services.  
> You must also provide your own **Mapbox key**, which can be obtained from the [Mapbox documentation](https://docs.mapbox.com/android/maps/guides/install/).
